Day-to-Day Work Planning

This program allow the user to enter the work planned to be done on that particular day and if the work has been done the user may cancel the work done
Please follow the instructions properly

Enter text into the input field to add items to your work plan list.

Press "ENTER" or Click the "complete" Button to mark it as complete.

Click the "X" to remove the item from your list.
